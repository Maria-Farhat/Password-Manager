# Password-Manager
A simple GUI-based password manager built with Python and Tkinter that securely stores and retrieves login credentials with JSON-based persistence.

# Features

*  Generate strong, random passwords
*  Save credentials (website, email, password)
*  Search for saved logins
*  JSON-based data storage (persistent)
*  User-friendly GUI built with Tkinter
*  Error handling for missing or incorrect data


# Built With
* Python 3
* Tkinter (GUI)
* JSON (data storage)

# Project Structure

```
password-manager/

```

# How to Run

1. Clone the repository:

```
git clone https://github.com/Maria-Farhat/password-manager.git
```

2. Navigate into the project folder:

```
cd password-manager
```

3. Run the app:

```
python main.py
```


 # How It Works

* The user enters a website, email, and password.
* Passwords can be generated automatically using Python’s random module.
* Data is stored locally in a JSON file.
* Users can search for saved credentials by website name.


# Disclaimer

This project is for educational purposes only and does not implement advanced encryption. Do not use it to store real sensitive data.
